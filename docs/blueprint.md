# **App Name**: PropConnect Mini

## Core Features:

- Listing Creation: Allow property owners to input details about their property, including address, price, and amenities.
- Media Upload: Enable owners to upload photos and videos of their properties to showcase them effectively.
- Search and Filtering: Enable the telegram mini app users to search for rental properties based on location, price range, and other criteria.
- Contact the Owner: Enable users to directly message property owners through the Telegram Mini App
- AI-Powered Property Description Enhancement: Use a tool to generate an enhanced description of property, by using AI to extract implicit details that could improve the listing.

## Style Guidelines:

- Primary color: Deep turquoise (#46B1A4) to create a balance of vibrancy and trust.
- Background color: Very light grayish-turquoise (#F2F9F9).
- Accent color: Teal (#468B7A) for a more vivid contrast and to highlight actions.
- Body and headline font: 'Inter', a sans-serif font for clean, modern readability.
- Use simple, outline-style icons to represent property types and amenities.
- Design a clean, card-based layout for property listings with clear visual hierarchy.
- Subtle transitions and loading animations to enhance user experience.
